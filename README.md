# 奥斯达通信官方网站

深圳市奥斯达通信有限公司企业官方网站。

## 网站功能

- 公司介绍与企业文化
- 荣誉资质与业务许可
- 人才招聘
- 企业通信服务介绍
- 客户案例
- 新闻发布与新闻分类
- 合作客户展示
- 客户留言表单
- 管理后台
- Supabase 数据库连接
- 全站搜索与暂无数据页面

## 项目技术

- HTML
- CSS
- JavaScript
- Supabase
- GitHub Pages
